IPAX surround comparison test

For each pair, the same text/background is shown twice — once isolated, once surrounded by simulated desktop chatter — but never at the same time. Each version fills the whole screen for a few seconds, in random order, separated by a neutral-gray pause so your eyes actually adapt to each condition before you judge. Afterward, say whether the first or second was more comfortable, or the same. This is a relative judgment (like an eye exam's "better, worse, or the same?"), a companion to the absolute rating in fragility-test.html, not a replacement for it.
